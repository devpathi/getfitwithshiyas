# How to update the website (no coding needed)

Everything below is done through the visual editor at **`https://your-site-address/admin`**.
Log in with the email + password you were invited with.

After you click **Publish** on any change, the website updates itself automatically in about
**1 minute**. Just refresh the page to see it live.

---

## ✍️ Add a new blog post

1. Go to **/admin** and log in.
2. Click **Blog Posts** → **New Blog Posts**.
3. Fill in the **Title**, a short **Description**, and the **Date**.
4. (Optional) Add a **Cover image**.
5. Write your post in the **Post content** box.
6. Leave **Draft** off to publish now (turn it on to hide it for later).
7. Click **Publish** (top right) → **Publish now**.

---

## 💬 Add or change a testimonial

1. **/admin** → **Testimonials**.
2. Click an existing one to edit it, or **New Testimonials** to add one.
3. Fill in the client's **name**, their **result** (e.g. "Lost 12 kg in 5 months"), and the
   **quote**. Add a **photo** if you have one.
4. Turn on **Show on homepage** to feature it on the front page.
5. **Publish**.

> The three placeholder testimonials are marked `[[REPLACE]]` — swap them for real ones.

---

## 💰 Change a price

1. **/admin** → **Pricing Packages**.
2. Click the package to edit.
3. Change the **Price** (e.g. `₹5,000`) and any **Features**.
4. **Publish**.

---

## ☎️ Change phone, WhatsApp, email or social links

1. **/admin** → **Site Settings** → **Contact & Site Info**.
2. Update any field. For **WhatsApp number**, enter digits only with country code
   (e.g. `918197081829`).
3. **Publish**. This updates the number/links everywhere on the site at once.

---

## 🏋️ Add or edit a program/service

1. **/admin** → **Programs / Services**.
2. Edit an existing one or create a new one. Set the **Mode** (In-person / Online / both).
3. **Publish**.

---

## 🖼️ Adding photos

When a field asks for an image, click **Choose an image → Upload** and pick the file from your
computer. To replace the big homepage and About photos (marked `[[REPLACE]]`), ask whoever set up
the site to drop the files into `public/images/` — or send them over and they'll add them.

---

## 📅 Bookings & 📨 Contact form

- **Bookings** are handled by **Calendly**. To change your available times, log into your Calendly
  account — the website calendar updates automatically. No website edit needed.
- **Contact form** messages arrive in **Netlify → Forms** (and by email if a notification is set
  up). Nothing to manage on the site itself.

---

## ❓ If something looks wrong

- Changes take ~1 minute to go live — wait a moment and refresh.
- If a change didn't appear, make sure you clicked **Publish → Publish now** (not just *Save*).
- For anything you can't do from `/admin`, contact whoever set up the site.
